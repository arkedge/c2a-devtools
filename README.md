# C2A DevTools

## ⚠️ THIS REPOSITORY HAVE BEEN ARCHIVED ⚠️

C2A DevTools は [Gaia](https://github.com/arkedge/gaia) に統合されました。

## How to use

### Prerequisites

- Rust
- Node.js (v18 or later)
- yarn (v1.22 or later)

### Run and open browser

```console
$ cargo run
```

Now you can see the URL in the console. Open it in your browser.
